# Adrin Shrestha Portfolio

Personal portfolio website for junior web developer, WordPress developer, React intern and SEO executive roles in Nepal.

## Built With
- HTML5
- CSS3
- JavaScript
- Responsive Design
- SEO meta tags
- Schema markup
## Customize Before Publishing
- Replace `your-email@example.com`
- Replace GitHub link
- Replace LinkedIn link
- Add your real project links
- Add `og-image.jpg` if you want social preview image

## Hosting Options
Recommended: Vercel, Netlify, GitHub Pages, or cPanel hosting.
