WCAG 2.1 AA Commitment

Accessibility Statement

Adrin Shrestha is committed to ensuring digital accessibility for people of all abilities. I continuously improve the user experience for everyone and apply the relevant web accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. This website is fully conformant with WCAG 2.1 Level AA.

Accessibility Features Implemented

  • Semantic HTML5 Structure: Using native landmark elements (<header>, <main>, <nav>, <footer>, <article>) to allow screen readers to navigate content accurately.
  • Full Keyboard Navigation: All interactive elements, links, buttons, and form components are fully accessible via keyboard Tab key navigation with visible focus indicators.
  • Color Contrast Compliance: All text and graphical elements meet or exceed WCAG 2.1 AA minimum contrast ratios (4.5:1 for standard text, 3:1 for large text).
  • ARIA Attributes & States: Proper aria-label, aria-expanded, and aria-hidden markup applied across interactive mobile menus, icons, and dynamic widgets.
  • Reduced Motion Support: CSS media queries respect operating system preferences for reduced motion (prefers-reduced-motion).
  • Scalable Typography: Text can be zoomed up to 200% without loss of content or functional layout breaks.
